# 未来规划

> 按优先级排列，待实现的功能清单。

---

## 优先级 P0 — 核心体验

- [ ] **穿透成长跟上防御**：确保玩家穿透能有效削减怪物防御
- [ ] **生命偷取实际生效**：验证独立化后的 15% 上限是否足够
- [ ] **速度属性实战效果**：实现先手/双动机制，让速度有存在感

---

## 优先级 P1 — 内容扩展

- [ ] **PVP / 竞技场**：玩家对战，支持天梯排名
- [ ] **宠物系统**：宠物可附属于装备，提供额外被动效果
- [ ] **公会系统**：创建公会、公会副本、公会战
- [ ] **符文系统**：独立于装备的强化系统，类似星穹铁道遗器

---

## 优先级 P2 — 玩法丰富

- [ ] **roguelike 玩法**：每次探索随机生成地图、技能、事件
- [ ] **世界 boss**：全服玩家共同挑战，伤害排名奖励
- [ ] **神秘商人**：随机出现的商店，出售稀有道具
- [ ] **离线收益优化**：更详细的离线时间奖励计算与展示

---

## 优先级 P3 — 体验优化

- [ ] **伤害数字动画**：战斗中伤害飘字效果
- [ ] **屏幕震动**：暴击/技能释放时增加反馈感
- [ ] **战斗日志优化**：彩色区分伤害类型，折叠冗余信息
- [ ] **音效系统**：背景音乐 + 战斗音效
- [ ] **主题商店**：可购买/解锁不同 UI 主题

---

## 优先级 P4 — 长期目标

- [ ] **多语言支持**：中英文切换
- [ ] **新手引导**：首次游玩时的操作指引
- [ ] **数据存档云同步**：登录账户后数据可跨设备同步
- [ ] **成就故事线**：每个成就解锁对应的背景故事片段
